This is a game heavily inspired from "The Exit 8" by KOTAKE CREATE. I made this game with Unity 3000. Very limited structure change were made compare to THE EXIT 8 as my goal here is mainly to replica the endless walkway machenic from the EXIT 8 which I found very interesting.
This game can be play on browser from A PC. Pressing ESC while the game is not maximized will terminate the game. Here is also alink to the github repo of this game if you want to download it and play it on local
